# TechlancePrep 💻

> **The High-Performance Engineering Command Center for Mastering DSA, System Design, and Behavioral Interview Tracks.**

TechlancePrep is an open-source technical preparation platform engineered for developers aiming to land roles at high-growth startups and FAANG companies. Re-architected with a premium warm-stone light aesthetic and equipped with real-time user telemetry, the platform serves as a career-advancement dashboard.

👉 **Explore the Platform Live:** [techlanceprep.in](https://techlanceprep.in)

---

## 🎯 Key Features

### 1. Curated Question Repositories
* **600+ Expert Challenges**: Spanning core Data Structures & Algorithms (DSA), System Design concepts, and Frontend/Backend architecture.
* **STAR-Method Behavioral Tracks**: Dedicated HR questions built to help developers structure answers using the Situation, Task, Action, and Result methodology.
* **Difficulty Segmentation**: Granular breakdown into **Easy**, **Medium**, and **Hard** tasks.

### 2. Command Center Analytics Dashboard
* **Curriculum Mastery Graphs**: Live completion trackers displaying your velocity across different tracks.
* **Intensity Analysis**: Real-time visual telemetry summarizing performance across difficulty levels.
* **Bookmark & Activity Logs**: Save questions for review and review recently completed challenges.

### 3. Cybernetic UI Redesign
* **Natural Light Mode**: Obsidian ink headers, soft warm stone canvases, and subtle card borders for maximum readability.
* **Compiler Loading Simulation**: An interactive page transition loader that mimics a boot sequence, displaying live step logs and a real-time progress bar.
* **Auto-Dismiss Notification Engine**: Success and error warning banners slide in and auto-dismiss after exactly 2 seconds.

### 4. Advanced SEO Architecture
* **Google Sitelinks Schema**: JSON-LD `SiteNavigationElement` configurations enabling inline navigation subpages directly inside search engine result listings.
* **Search Engine Name Alignment**: Structured `WebSite` metadata maps search results to the brand name *TechlancePrep* rather than raw text.
* **Crawling Stability**: Custom static SVG favicon serving structures and sitemaps optimized with `select_related('topic')` query joins.

## 🛠️ Technology Stack

* **Core Framework**: Python & Django (Server-Side Rendering Architecture)
* **Database Engine**: PostgreSQL (Neon Serverless Pooler Integration)
* **CSS & Design System**: Vanilla CSS, Tailwind CSS Utility Classes, FontAwesome Icons
* **Static Assets Server**: WhiteNoise (Compressed & Caching manifest storage)
* **Metadata & SEO**: JSON-LD Schema Graphs, XML Sitemaps, Robots.txt Redirect Routing
